# Automation-Scripts

A collection of automation scripts for weird, miscellaneous, and oddly specific tasks.

## About

This repository is a storage place for small automation scripts that handle all kinds of strange and unconventional jobs — the kind of tasks that are too quirky for a dedicated project but still worth automating.

## What's Inside

- Scripts that automate repetitive or unusual workflows
- Handy utilities for weird, one-off tasks
- Experiments in automation that don't fit anywhere else

## Usage

Browse the scripts, pick what you need, and run them. Each script is self-contained and focused on a specific task.

## Contributing

Have an automation for something weird? Feel free to add it!
